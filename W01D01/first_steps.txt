# Python Setup in Local System
Step 1. Download and install python 3.14.x from python.org and install it but
while installing there is path checkbox which you will have to tick for
setting up python path such that python command works perfectly.
Step 2. Check if your python is working fine in CMD/Terminal by
typing python --version which should return correct python version.

# Install and Setup Jupyter Notebook
Step 1. Test if pip path is set up automatically:-
pip --version

Step 2. Install Jupyter notebook with this command:-
pip install notebook

Step 3. Check if jupyter path is seu up automatically:-
jupyter --version

Step 4. Run the Jupyter notebook:-
python -m jupyter notebook

Step 5. Enjoy writing code in python
